## 🚀 快速开始

### 1. 克隆仓库
`ash
git clone https://github.com/mashenjun886/skill-lab.git
cd skill-lab
`

### 2. 查看技能文档
`ash
cd skills/text-processing
`

### 3. 运行技能
`ash
python your-skill.py
`
